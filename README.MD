## ✨ Features
* **Omnibox Integration:** Type the `build` keyword in the address bar to bypass traditional search results. No more navigating through menus! 
* **Champion Aliases:** Whether you type "Twisted Fate" or a common nickname like "TF", the extension knows exactly what you mean.
* **Curated Analytics Sites:** Choose from a variety of the top build analytics websites and set your favorite as the default destination.

## 🛠️ Installation & Usage

### 1. Installation
Install the "Champ Build Finder" extension directly from the [Chrome Web Store](http://chromewebstore.google.com/detail/) 

### 2. Choose your build analytics website
Once installed, click the extension icon in your browser toolbar. Use the dropdown menu to select your preferred analytics platform. Your choice will be saved automatically.

### 3. Start Searching
You're all set! Simply type `build` followed by a space and your champion’s name into your browser’s address bar. Press Enter to jump straight to their recommended build.

## ❓ Frequently Asked Questions

**Can I request new champion aliases?**<br>
Yes! I am constantly updating the alias database to make it smarter. 

**Can I suggest a new analytics site?**<br>
Absolutely. If there's a reliable analytics platform that isn't currently supported, let me know and I will look into adding it.

**Why do I need to type build before the champion's name?**<br>
The build keyword acts as a "trigger" that tells your browser to route the request through the extension rather than performing a standard Google search. This allows for near-instant navigation to your target page.

**Where can I fid the list of champion aliases?**<br>
You can view the full alias list in our [champions.json](https://github.com/cdmanning/Champ-Build-Finder/blob/main/app/champions.json) or via the Help page in the extension pop-up.
